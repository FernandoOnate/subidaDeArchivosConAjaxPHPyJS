Términos de uso y privacidad.

Del acceso al almacenamiento.

En dispositivos móviles.

En navegares web para dispositivos celulares es posible que para poder subir imágenes te pida acceso al almacenamiento del dispositivo.
En cualquier caso, ningún desarrollador de esta app tiene acceso al dispositivo en cuestión ni a sus documentos, el acceso se limita exclusivamente
al navegador web que el usuario esté utilizando.

En ordenadores o pc de sobremesa.

La app no tiene acceso al disco duro ni sus archivos, el acceso se limita al navegador web que se esté utlizando.

Del tratamiento de los archivos y datos del cliente.

Los archivos que se suban al programa no podrán ser borrados por el usuario.
Si este lo desea, puede contactarse con el administrador del hosting web para la solicitud de eliminación de archivos al siguiente correo: laesquinaverde37@gmail.com
ABSOLUTAMENE TODOS los archivos que el usuario suba, serán visibles para cualquier otro usuario que pueda visitar la app.

Los desarrolladores o desarrollador de esta app tendrán acceso a los datos básicos del cliente

Cliente es un término para referirse al navegador web, los datos antes mencionados hacen referencia a los headers
de la petición HTTP que se haga al backend.

Estos datos suelen ser:

  • Dirección ip pública.
  • Tipo de navegador web.
  • Versión del navegador web.
  • Fecha de la solicitud.

Entre otros similares.
